Travel Essentials List

A simple web application to help users create and manage a list of travel essentials for their trips. This project allows users to easily keep track of items they need to pack for travel.

Features

Add items to your travel essentials list.

Mark items as packed or unpacked.

Remove items from the list.

Responsive and easy-to-use interface.

Tech Stack

Frontend: React.js

Styling: CSS

State Management: React useState
